
Prequisites:
    * NodeJS v14.13.1 or higher
    * Git

To clone repository, go the one father directory of your own choice and type:

```powershell
> git clone Z:\git_repos\Julia_Desktop
```

To install type dependencies, type:

```powershell
> cd Julia_Desktop
> npm install
```
